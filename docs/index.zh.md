# 重整化群

这是关于**重整化群**（Renormalization Group）理论的自学笔记。为记录学习过程，将笔记整理成文章，并编写 [Python 代码](https://github.com/Liu-Zhihang/Renormalization-Group/tree/main/code) 以加深理解。

## 课程概述

重整化群提供了理解物理规律如何随尺度变化的系统框架——从临界现象与相变到量子场论。本笔记涵盖动机、统计力学基础、临界指数与标度律等内容。
